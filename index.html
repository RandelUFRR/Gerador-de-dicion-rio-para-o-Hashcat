Gerador de Dicionários para Hashcat

Sobre o Projeto

Este repositório contém o código para gerar dicionários de senhas personalizadas, ideais para uso com o Hashcat, uma ferramenta poderosa para recuperação de senhas.

Recursos

  • Configuração flexível para criar combinações de senhas.
  • Suporte para diferentes regras de geração.
  • Compatibilidade direta com o Hashcat.

Como Começar

Para usar o gerador de dicionários, siga os passos abaixo:

  1. Clone este repositório:
    2. git clone https://github.com/RandelUFRR/Gerador-de-dicion-rio-para-o-Hashcat.git
  2. Siga as instruções no arquivo README.md para configurar e rodar o script.
